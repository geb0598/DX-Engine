## 1. 게임 개요

### 1.1 게임 컨셉

**타이틀**: "LastRoll" (가칭)
**장르**: Top-Down Survival Shooter with Roguelike Elements

플레이어는 끝없이 몰려오는 적들을 상대하며 생존 시간을 최대한 연장하는 것이 목표입니다. 레벨업을 통해 3가지 공격 시스템(직선 발사, 유도 미사일, 궤도 방어)을 강화하며, 동적인 조명과 카메라 효과가 몰입감을 극대화합니다.

### 1.2 핵심 특징

- **3가지 공격 타입**: 수동 조준, 자동 타겟팅, 궤도 방어
- **동적 Camera System**: 전환, 흔들림, 페이드 효과
- **진보된 적 AI**: 체스 나이트 이동(EnemyA), 돌진 공격(EnemyB)
- **오브젝트 풀링**: 성능 최적화를 위한 Actor 재사용
- **무한 레벨**: Tile 래핑 시스템으로 끊김 없는 플레이 공간
- **점진적 난이도**: 적 스폰 간격이 시간에 따라 감소

### 1.3 게임 상태 흐름

```
Lobby (타이틀 화면)
  ↓ [W 키 입력]
Loading (3초 카운트다운 + 카메라 연출)
  ↓
Playing (메인 게임플레이)
  ↓ [플레이어 사망]
EndSequence (사망 연출)
  ↓
End (재시작 대기)
  ↓ [W 키 입력]
Lobby로 복귀
```

## 2. Camera System 아키텍처

FutureEngine의 Camera System은 **Editor Camera**와 **Game Camera** 두 가지 독립적인 서브시스템으로 구성됩니다.

### 2.1 시스템 개요

| 구분 | Editor Camera | Game Camera |
|------|---------------|-------------|
| **클래스** | `UCamera` | `APlayerCameraManager` |
| **목적** | 에디터 뷰포트 제어 | 게임플레이/PIE 카메라 관리 |
| **파일 위치** | `Engine/Source/Editor/Public/Camera.h` | `Engine/Source/Actor/Public/PlayerCameraManager.h` |
| **라인 수** | 127 lines (헤더) | 290 lines (헤더) |
| **입력** | WASDQE + 마우스 | 스크립트/게임 로직 제어 |
| **투영 모드** | Perspective, Ortho(6방향) | Perspective, Ortho |
| **특수 기능** | 프러스텀 컬링, 다중 뷰포트 | Modifiers, Blending, Shake, Transition |

### 2.2 Camera 업데이트 파이프라인

#### Editor Camera (UCamera)
```
UCamera::UpdateInput(DeltaTime)
  ├─ 마우스 회전 (RMB 드래그)
  ├─ WASD 이동 (Perspective)
  ├─ QE 상하 이동 (World Z축)
  └─ 스크롤 속도 조절 (1-100 units/sec)

UCamera::Update(Viewport)
  ├─ UpdateMatrixByPers() 또는
  └─ UpdateMatrixByOrth()
     → View/Projection Matrix 생성
```

#### Game Camera (APlayerCameraManager)
```
APlayerCameraManager::UpdateCamera(DeltaTime) [5단계 파이프라인]
  ├─ 1. UpdateViewTarget()
  │     → UCameraComponent에서 FMinimalViewInfo 추출
  │
  ├─ 2. UpdateBlending()
  │     → ViewTarget ↔ PendingViewTarget 보간
  │
  ├─ 3. ApplyCameraModifiers()
  │     → Shake, Transition 등 효과 적용
  │
  ├─ 4. UpdateFading()
  │     → 화면 페이드 효과 업데이트
  │
  └─ 5. UpdateCameraConstants()
        → View/Projection Matrix 생성 (GPU 전송용)
```

### 2.3 렌더링 통합

```cpp
// 렌더러에서 카메라 선택 로직 (의사 코드)
if (IsGameMode() && PlayerCameraManager != nullptr) {
    // PIE/Game 모드: Game Camera 사용
    FCameraConstants CameraData = PlayerCameraManager->GetCameraConstants();
} else {
    // Editor 모드: Editor Camera 사용
    FCameraConstants CameraData = EditorCamera->GetFViewProjConstants();
}

// GPU로 상수 버퍼 업로드 (cbuffer b1)
DeviceContext->UpdateSubresource(ConstantBufferViewProj, 0, nullptr, &CameraData, ...);
```

##### 1단계: UpdateViewTarget()
```cpp
void APlayerCameraManager::UpdateViewTarget()
{
    if (ViewTarget.Target == nullptr)
        return;

    // Target Actor가 CameraComponent를 가지고 있으면 사용
    if (ViewTarget.CameraComponent != nullptr) {
        ViewTarget.CameraComponent->GetCameraView(ViewTarget.POV);
    }
    // 없으면 Actor의 Transform 사용
    else {
        ViewTarget.POV.Location = ViewTarget.Target->GetActorLocation();
        ViewTarget.POV.Rotation = ViewTarget.Target->GetActorRotation();
        // 기본 FOV, AspectRatio 등 사용
    }

    // 현재 POV에 복사
    CachedPOV = ViewTarget.POV;
}
```

##### 2단계: UpdateBlending()
```cpp
void APlayerCameraManager::UpdateBlending(float DeltaTime)
{
    // 전환 중이 아니면 스킵
    if (ViewTargetBlendTimeRemaining <= 0.0f)
        return;

    // 블렌드 알파 계산
    ViewTargetBlendTimeRemaining -= DeltaTime;
    float BlendAlpha = 1.0f - (ViewTargetBlendTimeRemaining / ViewTargetBlendTime);
    BlendAlpha = Clamp(BlendAlpha, 0.0f, 1.0f);

    // 선형 보간 (모든 POV 필드)
    CachedPOV.Location = Lerp(ViewTarget.POV.Location,
                               PendingViewTarget.POV.Location,
                               BlendAlpha);

    CachedPOV.Rotation = Lerp(ViewTarget.POV.Rotation,
                               PendingViewTarget.POV.Rotation,
                               BlendAlpha);
    CachedPOV.Rotation.Normalize();  // Quaternion 정규화 필수

    CachedPOV.FOV = Lerp(ViewTarget.POV.FOV, PendingViewTarget.POV.FOV, BlendAlpha);

    // 전환 완료 시 ViewTarget 교체
    if (ViewTargetBlendTimeRemaining <= 0.0f) {
        ViewTarget = PendingViewTarget;
        PendingViewTarget.Target = nullptr;
    }
}
```

##### 3단계: ApplyCameraModifiers()
```cpp
void APlayerCameraManager::ApplyCameraModifiers(float DeltaTime)
{
    // 우선순위 정렬 (TODO: 매 프레임 정렬 비효율적 - 최적화 필요)
    ModifierList.sort([](UCameraModifier* A, UCameraModifier* B) {
        return A->GetPriority() > B->GetPriority();
    });

    // 각 Modifier가 POV 수정
    for (UCameraModifier* Modifier : ModifierList) {
        if (Modifier != nullptr && Modifier->IsEnabled()) {
            Modifier->UpdateAlpha(DeltaTime);          // 블렌드 상태 업데이트
            Modifier->ModifyCamera(DeltaTime, CachedPOV);  // POV 수정
        }
    }
}
```

##### 4단계: UpdateFading()
```cpp
void APlayerCameraManager::UpdateFading(float DeltaTime)
{
    if (!bFading)
        return;

    FadeTimeRemaining -= DeltaTime;

    // 페이드 진행도 계산
    float FadeProgress = 1.0f - (FadeTimeRemaining / FadeDuration);
    FadeAlpha = Lerp(FadeFromAlpha, FadeToAlpha, FadeProgress);

    // OverlayColor에 반영 (RGB + Alpha)
    CachedPOV.OverlayColor = FVector4(FadeColor.X, FadeColor.Y, FadeColor.Z, FadeAlpha);

    // 페이드 완료
    if (FadeTimeRemaining <= 0.0f)
        bFading = false;
}
```

```cpp
void APlayerCameraManager::UpdateCameraConstants()
```

### 4.1 UCameraModifier (Base Class)

#### 블렌드 상태 머신

```cpp
enum class ECameraModifierBlendMode
{
    Disabled,      // Alpha = 0.0 (비활성)
    BlendingIn,    // Alpha: 0.0 → 1.0 (페이드 인)
    Active,        // Alpha = 1.0 (완전 활성)
    BlendingOut,   // Alpha: 1.0 → 0.0 (페이드 아웃)
};

class UCameraModifier : public UObject
{
protected:
    ECameraModifierBlendMode BlendMode;
    float Alpha;                    // 현재 블렌드 강도 (0.0~1.0)
    float BlendInTime;              // 페이드 인 시간
    float BlendOutTime;             // 페이드 아웃 시간
    float Priority;                 // 우선순위 (높을수록 나중에 적용)

public:
    // 매 프레임 호출: Alpha 업데이트
    virtual void UpdateAlpha(float DeltaTime);

    // 매 프레임 호출: POV 수정 (파생 클래스에서 override)
    virtual void ModifyCamera(float DeltaTime, FMinimalViewInfo& InOutPOV) = 0;

    // 활성화/비활성화
    void Enable(bool bInBlendIn);
    void Disable(bool bInBlendOut);
};
```

#### Alpha 업데이트 로직

```cpp
void UCameraModifier::UpdateAlpha(float DeltaTime)
{
    switch (BlendMode)
    {
    case ECameraModifierBlendMode::BlendingIn:
        Alpha += DeltaTime / BlendInTime;
        if (Alpha >= 1.0f) {
            Alpha = 1.0f;
            BlendMode = ECameraModifierBlendMode::Active;
        }
        break;

    case ECameraModifierBlendMode::BlendingOut:
        Alpha -= DeltaTime / BlendOutTime;
        if (Alpha <= 0.0f) {
            Alpha = 0.0f;
            BlendMode = ECameraModifierBlendMode::Disabled;
        }
        break;

    case ECameraModifierBlendMode::Active:
    case ECameraModifierBlendMode::Disabled:
    default:
        // Alpha 변화 없음
        break;
    }
}
```

```cpp
    ECameraShakePattern Pattern;        // 진동 패턴
    float Frequency;                    // 진동 주파수 (Hz, Sine 전용)

    // Decay (감쇠) 제어
    bool bUseDecayCurve;
    FCubicBezierCurve DecayCurve;       // Bezier 커브 기반 감쇠

    // 런타임 상태
    float ElapsedTime;                  // 경과 시간
    bool bIsShaking;

    // 출력 스무딩 (프레임간 부드러운 전환)
    FVector PreviousLocationOffset;
    FQuaternion PreviousRotationOffset;
};
```

#### 진동 생성 알고리즘

##### Sine Pattern
```cpp
float Phase = ElapsedTime * Frequency * 2.0f * PI;
float SineValue = sin(Phase);

// Decay 적용
float DecayFactor = bUseDecayCurve
    ? DecayCurve.Evaluate(ElapsedTime / ShakeDuration)
    : smoothstep(0.0f, 1.0f, 1.0f - (ElapsedTime / ShakeDuration));

// 최종 오프셋
FVector LocationOffset = Forward * (SineValue * LocationAmplitude * DecayFactor);
float RotationOffset = SineValue * RotationAmplitude * DecayFactor;
```

##### Perlin Pattern
```cpp
// Perlin 1D Noise 함수 사용
float PerlinX = PerlinNoise1D(ElapsedTime * 10.0f + 100.0f);
float PerlinY = PerlinNoise1D(ElapsedTime * 10.0f + 200.0f);
float PerlinZ = PerlinNoise1D(ElapsedTime * 10.0f + 300.0f);

// -1~1 범위를 -Amplitude~Amplitude로 변환
FVector LocationOffset = FVector(PerlinX, PerlinY, PerlinZ) * LocationAmplitude * DecayFactor;

// 회전도 Perlin으로 생성
float PerlinPitch = PerlinNoise1D(ElapsedTime * 10.0f + 400.0f) * RotationAmplitude * DecayFactor;
float PerlinYaw = PerlinNoise1D(ElapsedTime * 10.0f + 500.0f) * RotationAmplitude * DecayFactor;
float PerlinRoll = PerlinNoise1D(ElapsedTime * 10.0f + 600.0f) * RotationAmplitude * DecayFactor;
```

##### Random Pattern
```cpp
// 매 프레임 무작위 값 (-1~1)
FVector RandomVec = FVector(
    RandomRange(-1.0f, 1.0f),
    RandomRange(-1.0f, 1.0f),
    RandomRange(-1.0f, 1.0f)
);

FVector LocationOffset = RandomVec * LocationAmplitude * DecayFactor;

// 회전도 무작위

    RandomRange(-1.0f, 1.0f),
    RandomRange(-1.0f, 1.0f),
    RandomRange(-1.0f, 1.0f)

```

#### ModifyCamera 구현

```cpp
void UCameraModifier_CameraShake::ModifyCamera(float DeltaTime, FMinimalViewInfo& InOutPOV)
{
    if (!bIsShaking)
        return;

    ElapsedTime += DeltaTime;

    // 지속시간 초과 시 종료
    if (ElapsedTime >= ShakeDuration) {
        StopShake();
        return;
    }

    // 패턴별 오프셋 계산
    FVector LocationOffset = CalculateLocationOffset();
    FQuaternion RotationOffset = CalculateRotationOffset();

    // 스무딩 (부드러운 프레임간 전환)
    LocationOffset = Lerp(PreviousLocationOffset, LocationOffset, 0.5f);
    RotationOffset = Lerp(PreviousRotationOffset, RotationOffset, 0.5f);

    // Alpha 블렌딩 적용
    LocationOffset *= Alpha;
    RotationOffset = FQuaternion::Slerp(FQuaternion::Identity, RotationOffset, Alpha);

    // POV에 오프셋 적용
    InOutPOV.Location += LocationOffset;
    InOutPOV.Rotation = InOutPOV.Rotation * RotationOffset;
    InOutPOV.Rotation.Normalize();

    // 다음 프레임을 위해 저장
    PreviousLocationOffset = LocationOffset;
    PreviousRotationOffset = RotationOffset;
}
```

```cpp
void UCameraModifier_Transition::ModifyCamera(float DeltaTime, FMinimalViewInfo& InOutPOV)
{
    if (!bIsTransitioning)
        return;

    TransitionElapsedTime += DeltaTime;

    // 전환 완료 체크
    if (TransitionElapsedTime >= TransitionDuration) {
        InOutPOV = TargetPOV;  // 정확히 목표 위치로
        bIsTransitioning = false;
        return;
    }

    // 0~1 진행도 계산
    float Progress = TransitionElapsedTime / TransitionDuration;

    // Bezier Curve 적용 (옵션)
    float BlendAlpha = bUseTimingCurve
        ? TimingCurve.Evaluate(Progress)
        : Progress;  // 선형

    // Alpha 블렌딩도 적용 (Modifier 자체 블렌드)
    BlendAlpha *= Alpha;

    // 모든 POV 필드 보간
    InOutPOV.Location = Lerp(StartPOV.Location, TargetPOV.Location, BlendAlpha);

    // 회전: Quaternion Lerp (TODO: Slerp 구현)
    InOutPOV.Rotation = Lerp(StartPOV.Rotation, TargetPOV.Rotation, BlendAlpha);
    InOutPOV.Rotation.Normalize();

    // FOV 보간
    InOutPOV.FOV = Lerp(StartPOV.FOV, TargetPOV.FOV, BlendAlpha);

    // Clip Plane 보간
    InOutPOV.NearClipPlane = Lerp(StartPOV.NearClipPlane, TargetPOV.NearClipPlane, BlendAlpha);
    InOutPOV.FarClipPlane = Lerp(StartPOV.FarClipPlane, TargetPOV.FarClipPlane, BlendAlpha);

    // Ortho Width 보간 (Ortho 모드 전용)
    InOutPOV.OrthoWidth = Lerp(StartPOV.OrthoWidth, TargetPOV.OrthoWidth, BlendAlpha);
}
```

```lua
function CountdownCoroutine()
    local CountTexts = {"3", "2", "1"}

    for i, Text in ipairs(CountTexts) do
        -- 화면 중앙에 큰 텍스트 표시
        LoadingText = Text
        LoadingTextAlpha = 1.0

        -- 1초 대기
        WaitForSeconds(1.0)

        -- 페이드 아웃
        local FadeTime = 0.3
        local StartTime = GetTime()
        while (GetTime() - StartTime) < FadeTime do
            local Progress = (GetTime() - StartTime) / FadeTime
            LoadingTextAlpha = 1.0 - Progress
            coroutine.yield()
        end
    end

    LoadingText = ""
end
```

```lua
    InactiveActors = {},      -- { ["AEnemy"] = {Actor1, Actor2, ...}, ... }
    ActorToNameMap = {}       -- { [UUID] = "AEnemy", ... }
}

function ActorPool:Get(ActorName)
    local Pool = self.InactiveActors[ActorName]

    -- Pool에 사용 가능한 Actor가 있으면 재사용
    if Pool and #Pool > 0 then
        local Actor = table.remove(Pool)
        self:ActivateActor(Actor)
        return Actor
    end

    -- 없으면 새로 생성
    local NewActor = World:SpawnActor(ActorName)
    self.ActorToNameMap[NewActor:GetUUID()] = ActorName
    self:ActivateActor(NewActor)
    return NewActor
end

function ActorPool:Return(Actor)
    local ActorName = self.ActorToNameMap[Actor:GetUUID()]
    if ActorName == nil then return end

    -- 비활성화
    self:DeactivateActor(Actor)

    -- Pool에 반환
    if self.InactiveActors[ActorName] == nil then
        self.InactiveActors[ActorName] = {}
    end
    table.insert(self.InactiveActors[ActorName], Actor)
end

function ActorPool:ActivateActor(Actor)
    Actor:SetCanTick(true)
    Actor:SetActorHiddenInGame(false)
    Actor:SetActorEnableCollision(true)
end

function ActorPool:DeactivateActor(Actor)
    Actor:SetCanTick(false)
    Actor:SetActorHiddenInGame(true)
    Actor:SetActorEnableCollision(false)
    Actor:StopAllCoroutines()  -- 모든 코루틴 중지
end
```

```lua
    -- 적이 탐지 영역에서 이탈

        if Enemy == OtherActor then
            table.remove(DetectedEnemies, i)

    HomingCooldownTimer = HomingCooldownTimer + dt

    -- 자동 발사
    if HomingCooldownTimer >= HomingCooldown then
        local Target = FindNearestEnemy()
        if Target ~= nil then
            ShootHomingProjectile(Target)
            HomingCooldownTimer = 0
        end
    end
end

function FindNearestEnemy()
    local PlayerPos = Owner:GetActorLocation()
    local NearestEnemy = nil
    local MinDistance = math.huge

    for i, Enemy in ipairs(DetectedEnemies) do
        -- 적이 살아있는지 확인
        local EnemyScript = Enemy:GetScriptComponentByName("UScriptComponent")
        if EnemyScript then
            local IsDead = EnemyScript:GetVariable("IsDead")
            if not IsDead then
                local Distance = (Enemy:GetActorLocation() - PlayerPos):Length()
                if Distance < MinDistance then
                    MinDistance = Distance
                    NearestEnemy = Enemy
                end
            end
        end
    end

    return NearestEnemy
```

```lua
    if MoveTimer >= MoveInterval then
        MoveTimer = 0
        StartPos = Owner:GetActorLocation()

        -- 다음 상태: Lifting

-- Lifting: 상승 (30% 시간)
function StateLifting(dt)
    StateTimer = StateTimer + dt
    local LiftDuration = MoveAnimDuration * 0.3

    if StateTimer >= LiftDuration then
        -- 목표 높이 도달
        CurrentState = EMoveState.Rotate
        StateTimer = 0
    else
        -- 상승 진행
        local Progress = StateTimer / LiftDuration
        local CurrentHeight = Progress * LiftHeight
        local NewPos = StartPos + Vector(0, 0, CurrentHeight)
        Owner:SetActorLocation(NewPos)
    end

-- Rotate: 플레이어 방향으로 회전
function StateRotate(dt)
    local PlayerPos = _G.PlayerData.PlayerPos
    local CurrentPos = Owner:GetActorLocation()
    local Direction = (PlayerPos - CurrentPos):Normalize()

    -- 목표 각도 계산
    local TargetAngle = math.atan2(Direction.Y, Direction.X) * (180.0 / math.pi)
    local CurrentAngle = Owner:GetActorRotation():ToEuler().Z

    -- 회전
    local AngleDiff = TargetAngle - CurrentAngle
    -- -180~180 범위로 정규화
    if AngleDiff > 180 then AngleDiff = AngleDiff - 360 end
    if AngleDiff < -180 then AngleDiff = AngleDiff + 360 end

    local RotationStep = RotationSpeed * dt

    if math.abs(AngleDiff) < RotationStep then
        -- 회전 완료
        Owner:SetActorRotation(QuaternionFromEuler(0, 0, TargetAngle))

        -- 이동 방향 저장
        MoveDirection = Direction

        CurrentState = EMoveState.Moving
        StateTimer = 0
    else
        -- 회전 진행
        local NewAngle = CurrentAngle + math.sign(AngleDiff) * RotationStep
        Owner:SetActorRotation(QuaternionFromEuler(0, 0, NewAngle))
    end
end

-- Moving: 수평 이동 (40% 시간)
function StateMoving(dt)
    StateTimer = StateTimer + dt
    local MoveDuration = MoveAnimDuration * 0.4

    if StateTimer >= MoveDuration then
        CurrentState = EMoveState.Landing
        StateTimer = 0
    else
        -- 수평 이동
        local Progress = StateTimer / MoveDuration
        local MoveDistance = Progress * StepDistance

        local NewPos = StartPos + MoveDirection * MoveDistance + Vector(0, 0, LiftHeight)
        Owner:SetActorLocation(NewPos)
    end
end

-- Landing: 하강 (30% 시간)
function StateLanding(dt)
    StateTimer = StateTimer + dt
    local LandDuration = MoveAnimDuration * 0.3

    if StateTimer >= LandDuration then
        -- 착지 완료
        local FinalPos = StartPos + MoveDirection * StepDistance
        FinalPos.Z = GroundZ  -- 정확한 바닥 높이
        Owner:SetActorLocation(FinalPos)

        -- 공격 체크
        local PlayerPos = _G.PlayerData.PlayerPos
        local Distance = (PlayerPos - FinalPos):Length()
        if Distance < 1.0 then
            Attack()
        end

        CurrentState = EMoveState.Idle
        StateTimer = 0
    else
        -- 하강 진행
        local Progress = StateTimer / LandDuration
        local CurrentHeight = (1.0 - Progress) * LiftHeight

        local CurrentPos = StartPos + MoveDirection * StepDistance + Vector(0, 0, CurrentHeight)
        Owner:SetActorLocation(CurrentPos)
    end
end
```

```lua
    CurrentVelocity = Lerp(CurrentVelocity, TargetVelocity, AccelerationCoeff * dt)

    -- 이동
    local NewPos = CurrentPos + CurrentVelocity * dt
    Owner:SetActorLocation(NewPos)

    -- 구르기 애니메이션 (회전)
    ApplyRollingAnimation(dt, CurrentVelocity)

    -- 공격 체크
    if Distance <= AttackDistance then
        Attack()
    end

    -- 플레이어를 지나쳤는지 체크 (거리 증가)
    if Distance > LastDistance then
        CurrentState = EMoveState.Decelerating
    end

    LastDistance = Distance
end

-- Decelerating: 감속
function StateDecelerating(dt)
    -- 속도 감소
    CurrentVelocity = Lerp(CurrentVelocity, Vector(0,0,0), DecelerationCoeff * dt)

    -- 이동
    local CurrentPos = Owner:GetActorLocation()
    local NewPos = CurrentPos + CurrentVelocity * dt
    Owner:SetActorLocation(NewPos)

    -- 구르기 애니메이션
    ApplyRollingAnimation(dt, CurrentVelocity)

    -- 정지 체크
    if CurrentVelocity:Length() < 0.5 then
        CurrentVelocity = Vector(0, 0, 0)
        CurrentState = EMoveState.Idle
    end
end
```

#### 구르기 애니메이션

```lua
function ApplyRollingAnimation(dt, Velocity)
    -- 속도에 비례하는 회전
    local Speed = Velocity:Length()
    local RotationRate = Speed * RollSpeedFactor  -- 도/초

    -- 이동 방향에 수직인 축으로 회전
    -- (간소화: Z축 회전으로 구르는 효과)
    local CurrentRot = Owner:GetActorRotation():ToEuler()
    CurrentRot.X = CurrentRot.X + RotationRate * dt  -- Roll 축 회전

    Owner:SetActorRotation(QuaternionFromEuler(CurrentRot.X, CurrentRot.Y, CurrentRot.Z))
end
```

### 10.3 Camera Matrix 생성

#### View Matrix (LookAt)

```cpp
FMatrix FMatrix::CreateLookAtLH(
    const FVector& CameraPosition,
    const FVector& TargetPosition,
    const FVector& UpVector)
{
    FVector Forward = (TargetPosition - CameraPosition).Normalize();
    FVector Right = UpVector.Cross(Forward).Normalize();
    FVector Up = Forward.Cross(Right);

    // Row-Major 구조
    FMatrix ViewMatrix;
    ViewMatrix.M[0][0] = Right.X;       ViewMatrix.M[0][1] = Up.X;      ViewMatrix.M[0][2] = Forward.X;     ViewMatrix.M[0][3] = 0.0f;
    ViewMatrix.M[1][0] = Right.Y;       ViewMatrix.M[1][1] = Up.Y;      ViewMatrix.M[1][2] = Forward.Y;     ViewMatrix.M[1][3] = 0.0f;
    ViewMatrix.M[2][0] = Right.Z;       ViewMatrix.M[2][1] = Up.Z;      ViewMatrix.M[2][2] = Forward.Z;     ViewMatrix.M[2][3] = 0.0f;
    ViewMatrix.M[3][0] = -Right.Dot(CameraPosition);
    ViewMatrix.M[3][1] = -Up.Dot(CameraPosition);
    ViewMatrix.M[3][2] = -Forward.Dot(CameraPosition);
    ViewMatrix.M[3][3] = 1.0f;

    return ViewMatrix;
}
```

#### Projection Matrix (Perspective)

```cpp
FMatrix FMatrix::CreatePerspectiveFovLH(
    float FovY,         // 수직 시야각 (라디안)
    float AspectRatio,  // Width / Height
    float NearZ,
    float FarZ)
{
    float YScale = 1.0f / tan(FovY * 0.5f);
    float XScale = YScale / AspectRatio;
    float Q = FarZ / (FarZ - NearZ);

    FMatrix ProjMatrix = FMatrix::Zero();
    ProjMatrix.M[0][0] = XScale;
    ProjMatrix.M[1][1] = YScale;
    ProjMatrix.M[2][2] = Q;
    ProjMatrix.M[2][3] = 1.0f;
    ProjMatrix.M[3][2] = -Q * NearZ;

    return ProjMatrix;
}
```

## 결론

FutureEngine의 Camera System은 **Editor Camera**와 **Game Camera** 두 개의 독립적인 서브시스템으로 구성되며, Game Camera는 **5단계 업데이트 파이프라인**을 통해 View Target 블렌딩, Modifier 효과 적용, 화면 페이드 등을 처리합니다.

### 주요 Camera 기능

1. **Camera Shake**: Sine, Perlin, Random 패턴을 통한 진동 효과, Bezier 커브 기반 감쇠
2. **Camera Transition**: FMinimalViewInfo 간 부드러운 전환, Bezier Easing 지원
3. **Screen Fade**: 색상 및 투명도 보간을 통한 화면 페이드
4. **Follow Camera**: 플레이어를 추적하는 Top-Down 카메라 (Lua로 제어)

### 게임플레이 통합

- **GameManager.lua**: 게임 시작 시 카메라 연출 (3초 전환 + 카운트다운)
- **Player.lua**: Follow 모드 토글, Camera Transition Test 통합
- **EnemyB.lua**: 공격 시 Camera Shake로 임팩트 강조

### 기술적 강점

- **Modifier Chain**: 우선순위 기반 효과 스택, 확장 가능한 구조
- **Bezier Curve**: EaseIn/Out/InOut 등 다양한 타이밍 커브
- **Lua 통합**: 스크립트에서 카메라 완전 제어 가능
- **행렬 시스템**: Row-Major/Row-Vector로 일관된 변환 파이프라인

이 Camera System은 Top-Down Shooter 게임에 최적화되어 있으며, 동적 조명, 카메라 효과, 플레이어 추적 등이 유기적으로 연동되어 몰입감 있는 게임플레이를 제공합니다.
